# 🚀 Employee Dashboard (React + AG Grid)

A modern and interactive Employee Dashboard built using **React**, **AG Grid**, and **Recharts** with a clean UI and powerful features.

---

## ✨ Features

* 🔍 Search employees instantly
* 🏢 Filter by department
* 📊 Interactive charts (Bar + Pie)
* 🌙 Dark / Light mode toggle
* 📈 Summary cards (Total, Active, Avg Salary)
* 📋 Advanced data table (sorting, filtering, pagination)
* 🎨 Premium UI with smooth animations

---

## 🛠️ Tech Stack

* React.js
* AG Grid
* Recharts
* React Icons
* JavaScript (ES6+)
* CSS (Inline Styling)
